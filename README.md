# BondCalc

Калькулятор для расчёта стоимости облигаций на основе биномиальной модели процентной ставки.

## Описание

Приложение реализует 10-периодную биномиальную модель для оценки (по дефолту):

- **Бескупонной облигации (ZCB)** — цена облигации с погашением через T лет
- **Форвардного контракта** — форвардная цена облигации с исполнением в момент времени t
- **Фьючерсного контракта** — цена фьючерса на облигацию с исполнением в момент времени k
- **Опциона Call** — цена американского опциона Call на фьючерс на облигацию

## Параметры модели

| Параметр | Описание                                | Диапазон |
| -------- | --------------------------------------- | -------- |
| **T**    | Срок модели (лет)                       | 1-100    |
| **k**    | Момент исполнения фьючерсного контракта | 0-T      |
| **t**    | Момент исполнения форвардного контракта | 0-T      |
| **r₀**   | Начальная процентная ставка (%)         | ≥ 0      |
| **E**    | Страйк опциона (%)                      | ≥ 0      |

## Как работает расчёт

1. Строится биномиальное дерево процентных ставок на T шагов
2. Методом обратной индукции вычисляется цена облигации ZCB_T
3. Рассчитывается форвардная цена как отношение цен облигаций
4. Вычисляется цена фьючерса (без дисконтирования)
5. Оценивается американский опцион Call на фьючерс с проверкой на досрочное исполнение
